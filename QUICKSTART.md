# 🚀 StudySaarthi Quick Start

**Get started in 5 minutes!**

## Prerequisites
- Python 3.8+ installed
- OpenAI API Key (get free $5 credits)

## 1️⃣ Install (1 minute)

```bash
# 1. Create virtual environment
python -m venv venv

# 2. Activate it
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt
```

## 2️⃣ Configure (2 minutes)

```bash
# 1. Get API key from: https://platform.openai.com/account/api-keys
# 2. Create .env file (copy from .env.example)
# 3. Add your OPENAI_API_KEY to .env
```

## 3️⃣ Run (1 minute)

```bash
# Navigate to backend folder
cd backend

# Start the server
python app.py
```

## 4️⃣ Use (1 minute)

Open browser → http://localhost:5000 → Enter a topic → Learn! 📚

## 🆘 Stuck?

1. **"Module not found"** → Did you activate venv? (Check for `(venv)`)
2. **"API key error"** → Add OPENAI_API_KEY to .env
3. **"Connection refused"** → Is Flask running in terminal?

See **ERROR_FIXES.md** for detailed troubleshooting.
